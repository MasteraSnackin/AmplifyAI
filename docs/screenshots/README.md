# Screenshots

This folder holds the images referenced in the main project [`README.md`](../../README.md).

To make the screenshots render on GitHub, save your image files here using these **exact filenames**:

| Filename | Screenshot shown in README |
| --- | --- |
| `results-quizzes.png` | Analysis Results — Interactive Quizzes (video + viral score + reasoning) |
| `quiz-answer-reveal.png` | Quiz answer reveal with correct/incorrect highlighting |
| `social-variations-ab.png` | Social Variations (A/B) side-by-side editor |
| `ad-scripts.png` | Multi-length Ad Scripts (15s / 30s / 60s) |
| `analytics-dashboard.png` | Performance Tracking analytics dashboard |

## How to add them

1. Save each screenshot into this folder with the matching filename above (PNG recommended).
2. Commit and push:
   ```bash
   git add docs/screenshots
   git commit -m "docs: add app screenshots"
   git push
   ```
3. The images will then render automatically in the README on GitHub.
